# 📌 Projeto TDD

Este projeto tem como objetivo aplicar os princípios de **Test-Driven Development (TDD)** no desenvolvimento de software, garantindo código mais confiável, testável e de fácil manutenção.

## 🧠 O que é TDD?

**TDD (Test-Driven Development)** é uma abordagem de desenvolvimento baseada em ciclos curtos:

1. Escrever um teste (que inicialmente falha)
2. Implementar o código mínimo para passar no teste
3. Refatorar o código mantendo os testes verdes

Esse ciclo é conhecido como:

> 🔴 Red → 🟢 Green → 🔵 Refactor

---

## 🚀 Objetivos do projeto

* Praticar a metodologia TDD na prática
* Garantir maior qualidade de código
* Desenvolver código orientado a testes
* Facilitar manutenção e evolução do sistema

---

## 🛠️ Tecnologias utilizadas

* Linguagem: *Java, Spring*
* Framework de testes: *JUnit*

---

## ✅ Exemplo de fluxo TDD

1. Criar um teste para uma funcionalidade
2. Executar os testes → falhar ❌
3. Implementar a funcionalidade
4. Executar os testes → passar ✅
5. Refatorar o código 🔄
